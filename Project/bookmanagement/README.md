# 도서 관리 프로그램 (Singly Linked List)

단순 연결 리스트 자료구조를 활용하여 도서를 추가/삭제/조회하고 전체 목록을 출력하는 콘솔 프로그램입니다. 입력값으로 책 번호, 제목, 저자, 출판 연도를 받아 저장하며, 책 번호는 중복될 수 없습니다.

## 요구 사항
- Python 3.8 이상
- 외부 라이브러리 없음

## 실행 방법
1) 터미널에서 프로젝트 루트로 이동한 뒤 아래 명령을 실행합니다.

```
cd Project/bookmanagement
python bookmanagement_app.py
```

## 메뉴 구성
- 1. 도서 추가: 책 번호, 제목, 저자, 출판 연도를 입력하여 추가합니다. 책 번호가 중복되면 오류 메시지를 출력합니다.
- 2. 도서 삭제 (책 제목으로 삭제): 입력한 책 제목과 일치하는 도서를 삭제합니다.
- 3. 도서 조회 (책 제목으로 조회): 입력한 책 제목과 일치하는 도서를 조회하여 상세 정보를 출력합니다.
- 4. 전체 도서 목록 출력: 저장된 모든 도서를 출력합니다. 도서가 없으면 안내 메시지를 출력합니다.
- 5. 종료: 프로그램을 종료합니다.

## 실행 예시
```
=== 도서 관리 프로그램 ===
1. 도서 추가
2. 도서 삭제 (책 제목으로 삭제)
3. 도서 조회 (책 제목으로 조회)
4. 전체 도서 목록 출력
5. 종료
메뉴를 선택하세요: 1
책 번호를 입력하세요: 100
책 제목을 입력하세요: 자료구조와 알고리즘 with 파이썬
저자를 입력하세요: 최영규
출판 연도를 입력하세요: 2023
도서가 추가되었습니다.

메뉴를 선택하세요: 4
현재 등록된 도서 목록:
[책 번호: 100, 제목: 자료구조와 알고리즘 with 파이썬, 저자: 최영규, 출판 연도: 2023]

메뉴를 선택하세요: 2
삭제할 책 제목을 입력하세요: 자료구조와 알고리즘 with 파이썬
도서가 삭제되었습니다.
```

## 구현 개요
- `Node`: 단순 연결 리스트 노드. `data`(Book)와 `link`를 보유.
- `LinkedList`: 삽입/삭제/탐색(`find_by_title`, `find_pos_by_title`, `find_by_id`) 제공.
- `Book`: 도서 데이터 클래스(책 번호, 제목, 저자, 출판 연도).
- `BookManagement`: 사용자 메뉴 처리 및 비즈니스 로직.

## 참고 사항
- 입력 처리에서 `.strip()`을 사용하지 않으며, `int()` 변환은 앞뒤 공백을 허용합니다.
- 데이터는 메모리에만 저장되며, 프로그램 종료 시 초기화됩니다.
