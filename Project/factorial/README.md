# 팩토리얼 계산기 (반복 vs 재귀 비교)

반복(iteration)과 재귀(recursion) 방식으로 n!을 계산하고, 필요 시 실행 시간을 비교하는 콘솔 프로그램입니다. 음수 입력은 허용하지 않으며, 0! = 1로 처리합니다.

## 요구 사항
- Python 3.8 이상
- 외부 라이브러리 없음

## 실행 방법
프로젝트 루트에서 다음 명령을 실행합니다.

```
python Project/factorial/factorial_app.py
```

## 메뉴 구성
- 1. 반복법으로 n! 계산
- 2. 재귀법으로 n! 계산
- 3. 두 방식 모두 계산 후 결과/시간 비교
- 4. 준비된 테스트 데이터 일괄 실행(여러 n에 대해 반복/재귀 계산 및 시간 출력)
- q. 종료

## 사용 예시
```
================ Factorial Tester ================
1 반복법으로 n! 계산
2 재귀법으로 n! 계산
3 두 방식 모두 계산 (결과/시간 비교)
4 준비된 테스트 데이터 일괄 실행
q) 종료
--------------------------------------------------
선택 : 3
n(정수, 0 이상)을 입력하세요: 10
[반복] 10! = 3628800
[재귀] 10! = 3628800
결과 일치 여부: 일치
[반복] 시간: 0.0000 s |   [재귀] 시간: 0.0000 s
```

## 주의 사항
- 매우 큰 n에서 재귀 호출은 파이썬 최대 재귀 한계를 초과할 수 있습니다. 이 경우 RecursionError가 발생할 수 있습니다.
- 음수 입력 시 `ValueError`를 발생시키거나 오류 메시지를 출력합니다.

## 파일
- `factorial_app.py`: 반복/재귀 방식 구현 및 메뉴 인터페이스
